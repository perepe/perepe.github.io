 Installation:

- Run the executable in Timeless\Binaries\Win32\Timeless.exe

- If execution fails, run the installer UE4PrereqSetup_x86.exe and then run the game again.


 Controls:

- WASD        -> movement
- spacebar    -> jump
- right-click -> capture time
- left-click  -> time jump
- ESC / Q     -> ingame menu
- 0,1,2,3     -> graphics quality (0: worst, 3: best)


 Tips:

- If the game runs slow, try adjusting the graphics quality with buttons 0-3 (0: worst, 3: best)
